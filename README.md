# Tencocoa for Android (a.k.a. Kbtter7)
The hopeful &amp; extreme twitter client for Android.

## 概要
Android用のKbtterです。機能的にはKbtter4みたいな感じです。
Tencocoaという名前は言わずもがな比那名居「天子」と保登「心愛」から。

## 予定されている機能
* Streaming APIの対応
* API取得の半強制
 - StarryEyesと同じ姿勢
* スキン機能
* マテリアルデザイン
